# RTT-Trend-Aware CUBIC in ns-3

Comprehensive CSE322 project repository for evaluating TCP congestion control in ns-3:

- CUBIC recreation: tcp-cubic-custom
- RTT-trend-aware variant: tcp-cubic-rtt-aware

The project includes wired and wireless simulations, parameter sweeps, automation scripts, plotting pipelines, and report artifacts for reproducibility.

## What This Project Covers

- Implements and evaluates RTT-trend-aware CUBIC behavior in ns-3.
- Compares tcp-cubic-custom vs tcp-cubic-rtt-aware across multiple scenarios.
- Runs wired (CSMA), wireless (802.11 with mobility), and bonus hybrid cross-transmission experiments.
- Produces metrics and plots for throughput, delay, PDR, DRR, jitter, fairness, and related outputs.
- Includes reproducible report generation assets.

## Quick Start

### 1) Clone and enter the project

```bash
git clone <your-repo-url>
cd CSE-322-NS-3-Project
```

### 2) Create virtual environment and install Python dependencies

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install numpy matplotlib pandas seaborn cmake-build-extension
```

### 3) Configure and build ns-3

```bash
./ns3 configure
./ns3 build
```

### 4) Sanity run

```bash
./ns3 run "scratch/cubic-networks-simulation --networkType=wired --numNodes=20 --numFlows=10 --pktPerSec=100 --coverageMultiplier=1"
```

You should see a CSV_ROW line in stdout for a successful run.

## Reproduce Main Evaluation Outputs

### A) Full evaluation reproduction (single command)

```bash
bash reproduce-evaluation.sh
```

This script rebuilds and reproduces the base comparison outputs and plots.

### B) Viva sweep and 25-plot pipeline

Run sweep:

```bash
python run-viva-network-sweep.py --build
```

Generate plots:

```bash
python plot-viva-metrics.py
```

Default outputs are written under network_results and plot files generated by the script.

### C) Bonus hybrid experiments

Run hybrid simulations:

```bash
python run-bonus-simulations.py
```

Generate bonus plots:

```bash
python plot-bonus-results.py
```

Bonus outputs are stored under bonus_results.

## Troubleshooting

### ns-3 says "You need to configure ns-3 first"

Run:

```bash
./ns3 configure
./ns3 build
```

### Plot script fails with missing Python module

Activate venv and install dependencies:

```bash
source .venv/bin/activate
python -m pip install numpy matplotlib pandas seaborn
```

### seventh-packet-byte-count.sh fails

Install gnuplot:

```bash
sudo apt install gnuplot
```

### test.py reports "0 of 0 tests passed"

Tests are disabled by default in current configuration. Enable tests with:

```bash
./ns3 configure --enable-tests
./ns3 build
./test.py
```
